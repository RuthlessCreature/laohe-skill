# 创意工程与跨界点子参考

Use this reference when the user asks for 骚点子, creative product ideas, unusual engineering applications, cross-domain combinations, growth hacks, monetization angles, or new product wedges.

## Core Rule

先发散，再收敛。老何的创意不是随机脑暴，而是把工程能力、市场缺口、商业模式、渠道优势和人性动机做组合爆破。

## 老何骚点子评分

Score every idea from 1-5:

```latex
IdeaScore =
0.22D + 0.18M + 0.16F + 0.14V + 0.12S + 0.10R + 0.08X
```

Where:

- \(D\): distribution access.
- \(M\): monetization speed.
- \(F\): feasibility.
- \(V\): value intensity.
- \(S\): surprise or memorability.
- \(R\): repeatability/system leverage.
- \(X\): cross-domain novelty.

Reject ideas with high novelty but no buyer, no trigger, or no cheap validation.

## Useful Patterns

- Spreadsheet-to-SaaS: turn an ugly but mission-critical Excel workflow into a hosted tool with import/export compatibility.
- Audit-to-product: sell a manual diagnostic first, then productize repeated findings.
- Workflow arbitrage: find teams moving data between email, PDF, Excel, and portals; automate the boring bridge.
- Compliance wedge: turn regulation fear into checklist, evidence collection, and reporting software.
- Data exhaust product: collect operational traces during service delivery, then sell benchmark insights.
- Agent-in-the-loop: automate 70% of expert work while keeping human approval for liability, taste, or trust.
- Physical-digital bridge: combine camera, sensor, QR, NFC, or mobile capture with backend workflow automation.
- Template virality: publish a useful spreadsheet/template, then upsell hosted automation.
- Internal tool externalization: package a sharp internal operator tool as a niche SaaS.
- Outcome packaging: sell "reduce X by Y%" instead of software seats when value is measurable.

## Output Shape

For ideation tasks, produce:

- 10-20 raw ideas if the user asks broadly.
- 3-5 shortlisted ideas with score, buyer, mechanism, prototype, monetization, validation test, and main risk.
- 1 recommended wedge with a 48-hour experiment.

## Anti-Patterns

- Clever demo with no buyer.
- AI wrapper with no workflow ownership.
- Engineering challenge that does not unlock distribution or margin.
- Marketplace idea with no supply acquisition path.
- Consumer novelty with weak retention.
- Enterprise idea with no compliance, procurement, or integration story.
